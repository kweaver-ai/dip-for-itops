# DIP for ITOps (AI运维大脑)

## 简介
DIP for ITOps （Data Intelligence Platform for IT Operations，IT智能运维）是基于DIP平台构建、专注于 ‌IT运维智能化‌的AI智能体应用集合，旨在通过知识驱动和AI增强的运维智能体，实现IT运维的自动化、智能化与决策优化。其通过整合多源异构机器数据和机器数据统一治理，基于运维知识网络和运维智能体，实现告警事件高效分析、故障根因快速定位和修复、应用健康分析评估、机器数据智能查询等I场景的智能运维，帮助运维人员提升IT运维效率、保障业务连续性

## 关键价值

- **消除机器数据孤岛**：DIP for ITOps通过数据虚拟化技术实现多源异构机器数据的整合，消除机器数据孤岛。
- **高效关联分析**： 通过运维知识网络实现从机器数据到运维知识的编织，实现机器数据的高效关联分析。
- **运维辅助决策**： 基于AI大模型和运维知识网络, 通过运维智能体辅助运维人员进行运维决策，实现“感知-分析-决策-执行”闭环 ，弥补运维人员个人经验不足。
- **场景化应用可扩展**：内置丰富的运维智能体、观测对象模型和机器数据模型，覆盖主要的IT运维场景。同时持场景应用快速扩展，可根据场景需求灵活配置扩展。

## 场景应用

### 1. 告警与故障分析 (ITOps Alert Analysis)
告警与故障分析是专为运维人员打造的排障利器。应用提供智能告警收敛与故障溯源分析能力，能够自动将离散告警收敛为故障点，并聚合为完整的运维问题。包括如下关键能力：
- **告警收敛**：把告警收敛并聚合为故障，评估故障影响范围与影响程度。
- **溯源分析**：还原故障发生过程。
- **根因定位**：高效定位故障根因。
